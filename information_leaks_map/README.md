# Система мониторинга утечек

## Описание
Система мониторинга утечек — это инструмент для отслеживания и анализа утечек данных. Он предоставляет функционал для мониторинга утечек, проверки нарушений безопасности и генерации отчетов.

## Установка

### Требования
- Python 3.10+
- Django 5.2+
- pip

### Установка
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Создайте файл `.env` в корневой директории проекта и добавьте туда API-ключи и другие конфигурационные переменные.

3. Примените файл `.env.example` из директории `utils` для примера конфигурации.

4. Запустите Django-проект:
   ```bash
   python manage.py runserver
   ```

## Использование

### Запуск сервера
Запустите сервер с помощью команды:
```bash
python manage.py runserver
```

### Тестирование
Запустите тесты с помощью команды:
```bash
pytest
```

### Генерация отчетов
Сгенерируйте отчеты с помощью команды:
```bash
python manage.py generate_report
```

## Конфигурация
Конфигурационные файлы находятся в директории `information_leaks_map`. Основные файлы:
- `settings.py`: Основные настройки Django.
- `urls.py`: Маршруты URL.
- `api_client.py`: Клиент для взаимодействия с API.
